Frequently Asked Questions

To add a new user, navigate to the 'Users' section in the admin panel. Click the 'Add User' button, fill in the required details (name, email, role), and submit the form. An invitation will be sent to the new user's email address.
Our platform supports multiple user roles, including Administrator, Editor, and Viewer. Each role has a specific set of permissions that control access to different features and data. You can find a detailed breakdown in the 'User Management' documentation.
As an administrator, you can reset a user's password from the 'Users' section. Find the user, click on the 'Edit' action, and select the 'Send Password Reset' option. The user will receive an email with instructions to set a new password.
